## 📌 Project Overview

This application uses a trained K-Nearest Neighbors (KNN) model to predict whether a patient has a high or low risk of heart disease.

The user enters medical details, and the system:

- Preprocesses the input
- Applies feature scaling
- Makes prediction using trained model
- Displays risk result with probability

---

## 🧠 Machine Learning Details

- **Algorithm:** K-Nearest Neighbors (KNN)
- **Preprocessing:** One-Hot Encoding
- **Feature Scaling:** StandardScaler
- **Model Saving:** Joblib
- **Deployment:** Streamlit Cloud
